RideCircle – Ride Sharing Android App

Overview:
RideCircle is an Android-based ride-sharing application that connects drivers and passengers for efficient and affordable travel. The app enables users to offer rides or find rides in real-time with a smooth and user-friendly experience.

Features:
🔐 User Authentication using Firebase OTP
🚘 Offer Ride & Find Ride functionality
📍 Google Maps integration for location tracking
📅 Ride scheduling and booking system
💳 Payment integration (Razorpay - planned/implemented)
📩 Real-time notifications (planned/using APIs)

Tech Stack:
Frontend: Android (Java, XML)
Backend/Services: Firebase Authentication
APIs: Google Maps API

How to Run the Project:
Clone the repository:
git clone https://github.com/your-username/RideCircle.git
Open in Android Studio
Sync Gradle
Run the app on emulator or device
